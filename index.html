Global E-Library

Pre-Launch Preparations: Beta version of the app with more features will be shared with language experts who can appprove the content.

Step-1: User will download the app and register with personal details.

Step-2: User will take the picture or upload from camera roll of the literature they want to read.

Step-3: The app will check the same in it's database if it is already there will provide the translation to the user. If it is not in the database, will forward the same to the respective language expert.

Step-4: The language expert will convert the literature and send it for approval to the admin of the app.

Step-5: Once admin approves the translation, the same is added to the database and the user will be notified.
